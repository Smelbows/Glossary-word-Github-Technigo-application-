Git versus Github

Git is an example of a Version Control System (VCS), which enables programmers to work simultaneously on a project and then bring together their separate additions in a controlled and logical way.
Github is a site that hosts Git repositories (although there are alternatives). Github also provides a number of tools and features (like forking and pull requests) to make working with a Git repository, and with a team of programmers, more straightforward.

Github features

  • Forking: Copying a repository from one Github account to another.
  • Pull requests and Merging: Making a request for your code changes get added to the original. The original creator then chooses to merge.
  • Issue Tracking: Used for bug reporting and feature requests.
